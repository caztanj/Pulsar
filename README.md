# Pulsar JavaScript Engine

A JavaScript engine written entirely from scratch. This means we feature our own implementation of a JavaScript VM that is **not** dependent on V8, SpiderMonkey, or JavaScriptCore.

The aim of this project is to build a highly capable JavaScript environment, designed both for running JavaScript standalone (similar to Node.js) and for embedding into other projects.

## Usage

You can run JavaScript files using the Pulsar executable. By default, scripts run in a restricted environment without access to the network or filesystem.

`Pulsar [options] <file.js>`

### Command Line Arguments

- `-network` - Grants the script permission to make network requests.
- `-filesystem` - Grants the script permission to interact with the local filesystem.
- `-print_ast` - Prints the generated Abstract Syntax Tree (AST) after parsing.
- `-print_bytecode` - Prints the generated bytecode before executing it in the VM.
- `-repl` - Starts an interactive Read-Eval-Print Loop (REPL).

## Building and Testing

You can build Pulsar using the Jai compiler. By default, running `jai first.jai` builds the executable in debug mode.

You can customize the build process by passing compile-time flags to `first.jai`:

- `release` - Builds the project with compiler optimizations enabled and disables the memory debugger.
- `test` - Compiles and runs the internal Pulsar test suite to verify core engine functionality.
- `test262` - Compiles and runs the official ECMAScript Test262 compliance suite (requires initializing the git submodule via `git submodule update --init`).

Example:
```bash
jai first.jai - release
jai first.jai - test262
```

## Modules

While building this JavaScript environment, we also developed several independent modules. Each module can be used standalone from this project and is easily embeddable into other Jai projects.

- **Http_Client** - Make HTTP requests in Jai. Supports both blocking and concurrent operations.
- **Http_Server** - An HTTP server for Jai. Implements HTTP/1.1, WebSockets, Server-Sent Events, and TLS.
- **JavaScript** - The core JavaScript bytecode engine.
- **LibreSSL** - Jai bindings for LibreSSL.
